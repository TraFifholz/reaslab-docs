# Projects & Imports

This guide covers creating a fresh Lean workspace, importing existing repositories, and joining shared projects via links.

## Create a new project

1. From the dashboard, either choose **New Project** for a blank workspace, or select a template to import as your starting point.
2. If you chose a blank project, enter a project name and select the Lean toolchain version to target.
3. Finish setup to open the IDE with the default project layout.

## Import from Git

![Import Git](/images/import-git.png)

1. Paste any Git URL on the import page for manual setup.
2. If you sign in with GitHub, your repositories appear automatically; otherwise, link GitHub in Account Settings.

## Join via Share Link

- If you're joining someone else's project and have a share link, sign in to your account and open the link to create and enter the project with the appropriate permissions.
