# Markdown Preview

Reaslab supports Markdown alongside Lean so you can document projects and publish explanations without leaving the IDE.

## Live preview

![Markdown Live Preview](/images/markdown-live-preview.png)

- Open any `.md` file to see a live preview in the right sidebar alongside the editor.
- Scrolling stays synchronized between the editor and the preview so you can keep context.

## Math and code blocks

- Inline and block math render automatically using standard LaTeX syntax.
- Code fences highlight popular languages such as Python and JSON for quicker skimming.

Use Markdown files to capture project notes, proof outlines, or onboarding instructions right next to your Lean sources.
