# Math Modeling Agent

The Math Modeling Agent is designed to assist you in solving complex mathematical modeling problems, analyzing data, and generating professional PDF reports based on competition templates.

## Accessing the Agent

After creating and entering a [project](/guides/projects.md), select the first option, **Math Modeling Agent**, from the LLM list in the bottom chat panel to enter the interaction mode.

## Basic Features
The Math Modeling Agent provides specific functionalities to simplify your workflow:
- **Template Generation**: Automatically sets up the project structure for mathematical modeling.
- **Context Awareness**: Seamlessly loads the competition problem and data files via "Add File to Chat".
- **Report Generation**: Capable of outputting a fully formatted LaTeX-based PDF report at the end of the task.

## Example Workflow

1. On the template page, select the "Math Modeling Contest" category, filter to your target problem, and review its details.
2. Click **Use Template** to create your own project from that problem template.
3. After entering the project, add the target problem file, such as `Problem.md`, and any related data files by right-clicking them and selecting **Add File to Chat**.
4. Switch the agent in the lower-right chat area to **Math Modeling Agent**.
5. Send a prompt such as:
   > "Solve the modeling problem in Problem.md. The data files are in the attachments folder. Please generate the final complete modeling report in PDF."
6. The agent will guide the modeling process and eventually generate the final report.
