# LaTeX Editor

ReasLab provides a powerful LaTeX editing experience with real-time feedback and integrated PDF preview.

## Overview

![LaTeX Editor Overview](/images/latex-overview.png)

- Edit LaTeX files with full syntax highlighting and auto-completion.
- View the compiled PDF side-by-side with your source code.

## Outline

![Document Outline](/images/tex-outline.png)

- Navigate huge documents easily with the outline view.
- Jump to sections, subsections, and labels with a single click.

## Real-time diagnostics

![Inline Diagnostics](/images/tex-inline-diagnostics.png)

- Inline error checking highlights issues directly in the editor as you type.
- Quickly identify syntax errors without waiting for a full compilation.

## Live compilation

![Compilation Output](/images/tex-compilation-live-output.png)

- The editor can compile your document automatically in the background.
- View live compilation logs to monitor the build process.

## Diagnostics panel

![Diagnostics Panel](/images/tex-compilation-output-diagnostics.png)

- Access a structured list of all warnings and errors in the diagnostics panel after a compilation finished.
- Click on an error to jump directly to the relevant line in your code.

## Hover information

![Hover Info](/images/tex-hover-info.png)

- Hover over symbols and commands to view documentation and usage details.

## Settings

![TeX Settings](/images/tex-settings.png)

- Configure the specific TeX engine (e.g., pdflatex, xelatex) for your project.
- Set the main document to ensure correct compilation even when editing included files.
