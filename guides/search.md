# Search

Locate proofs, tactics, and references quickly using the search tools built into ReasLab.

## In-file search

![In-file Search Panel](/images/infile-search-panel.png)

- Open the in-file panel from the editor toolbar or with `Cmd/Ctrl + F`.
- Toggle options for case sensitivity, whole-word matching, and regular expressions.
- Navigate through results without closing the panel to apply edits rapidly.

## Project-wide search

![Project-wide Search](/images/search-project.png)

- Switch the left sidebar to **Search** for cross-project lookups.
 - Provide include or exclude patterns when you want to narrow the scope.
- Use replace to update matching results across multiple files.

## Semantic search

![Semantic Search](/images/search-semantic.png)

- Open the **Semantic** view in the left sidebar to explore results ranked by meaning instead of exact text.
- Start semantic search from a selection inside the editor via the context menu when you need related lemmas or definitions.
