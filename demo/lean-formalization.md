# Lean Formalization and Machine-Checked Proofs

Bridge the gap between informal mathematics and formal verification by automatically translating natural language statements into machine-checked Lean proofs.

## Key Features

- **Natural Language Specification**: State mathematical problems in natural language and specify formalization requirements.

- **Automatic Translation**: Translate informal statements into precise Lean propositions automatically.

- **Clarifying Annotations**: Insert comments and annotations to clarify the formalized definitions.

- **Automated Proof Generation**: Generate a complete, compilable Lean proof without manual proof scripting.

- **Machine Verification**: Verify correctness through Lean's type checker and proof engine.
