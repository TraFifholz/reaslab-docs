# Markdown 预览

ReasLab 完美兼容 Lean 以及 Markdown 的联合生态。因此您可以在不离开 IDE 情况下，记录项目架构、发布注释说明或撰写普通学术文章。

## 实时预览

![Markdown Live Preview](/images/markdown-live-preview.png)

- 打开任意 `.md` 后缀文件，右侧边栏会自动渲染基于文本的无缝实时预览。
- 预览框的滚动条将与纯文本编辑器自动双向绑定与同步交互，确保时刻聚焦关键段落上下文。

## 数学公式系统与代码块

- 利用标准的 LaTeX 逻辑和符号段，能够自动渲染内联 (Inline) 及区块级别的精准数学公式。
- 利用格式围栏高亮包括 Python、JSON 以及 Lean 自身的各项语言代码。

我们提倡巧妙利用 Markdown 来存放您的项目备忘录、定理大纲或者为随后的 Lean 实现提供新手上路指导。
