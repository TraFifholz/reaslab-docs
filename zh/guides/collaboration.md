# 协作

Lean 项目很少是孤立进行的。ReasLab 支持同步编辑和审查流程，使团队在保持严谨推理的同时快速推进。

## 邀请队友

![Share Project with collaborators](/images/share-dialog.png)

- 点击顶部栏中的 **Share** 打开分享对话框。
- 根据您需要的访问级别，生成具有查看 (View)、编辑 (Edit) 或管理 (Manage) 权限的链接。
- 在同一个对话框中，还可以直接更新现有成员并调整他们的权限角色。

## 实时会话

![Online Peers](/images/online-peers-panel.png)

- 同一项目中的参与者的编辑内容会瞬间双向同步。
- “Online Peers (在线协作者)” 面板列出了当前连接到特定文件或整个工作区的所有人。
- 当需要进行证明结对编程时，点击协作者的头像即可“跟随”，将视口瞬间跳跃至对方光标处。

## 保持一致进度

![Collaborators](/images/collaborators-presence.png)

- 编辑器内部会显示协作者的光标与选区状态，体现存在感。
- 随时查看项目 History（历史记录），审查过往快照、恢复早期版本或提供异步的版本审阅意见。
