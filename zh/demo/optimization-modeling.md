# 优化建模与求解器集成

将优化问题的自然语言描述转换为可执行模型，辅以自动化求解器集成和专业文档输出。

## 核心功能

- **自然语言描述**：以自然语言描述真实世界的优化问题。
- **结构识别**：让 ReasLab 识别底层问题结构，如资源配置或路径优化。
- **一键代码生成**：一键生成完整可执行的 Python 优化脚本。
- **自动化文档**：自动导出结构化 LaTeX 文档和精美的 PDF 报告。
