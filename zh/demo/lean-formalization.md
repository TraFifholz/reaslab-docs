# Lean 形式化与机器验证证明

通过将自然语言陈述自动转换为机器可查验的 Lean 证明，弥合非形式化数学与形式化验证之间的鸿沟。

## 核心功能

- **自然语言规约**：以自然语言陈述数学问题，明确形式化需求。
- **自动转换**：将非形式化陈述自动翻译为精确的 Lean 命题。
- **澄清注解**：插入注释和注解以阐明形式化定义。
- **自动化证明生成**：无需手动编写证明脚本，即可生成完整可编译的 Lean 证明。
- **机器验证**：通过 Lean 的类型检查器和证明引擎验证正确性。
