# Keyboard shortcuts

The following keyboard shortcuts help you navigate and work efficiently in ReasLab. All shortcuts adapt to macOS (`Cmd`) and Windows/Linux (`Ctrl`) modifiers.

## Global navigation

- `Cmd/Ctrl + K`: Open command palette
- `Cmd/Ctrl + Shift + E`: Toggle file explorer
- `Cmd/Ctrl + F`: Open in-file search
- `Cmd/Ctrl + Shift + F`: Open project-wide search
- `Cmd/Ctrl + Shift + Space`: Toggle symbols palette

## Panel management

- `Cmd/Ctrl + B`: Toggle left sidebar
- `Cmd/Ctrl + J`: Toggle right sidebar

## File operations

- `F2`: Rename file or folder
- `Alt/Option + N`: Create new file
- `Cmd/Ctrl + Alt/Option + N`: Create new folder
- `Del`: Delete selected file or folder

## Tab management

- `Alt/Option + W`: Close current tab
- `Cmd/Ctrl + Alt/Option + W`: Close all tabs
- `Alt/Option + Shift + W`: Close current project

## Editor appearance

- `Cmd/Ctrl + Shift + .`: Increase font size
- `Cmd/Ctrl + Shift + ,`: Decrease font size

## Command palette

![Command Palette](/images/command-palette.png)

The command palette provides quick access to actions, snippets, and Lean utilities without leaving the keyboard.

### Open the palette

- Press `Cmd/Ctrl + K` to open the palette anywhere in the workspace.
- Type to filter commands; use the arrow keys to navigate results.
- Hit `Enter` to run the selected action.

### Common commands

- **File operations**: Create, rename, delete files and folders
- **Panel toggles**: Show or hide file tree, search, and Infoview panels
- **Navigation**: Jump to files, symbols, and definitions
- **Search**: Access different search modes (text, semantic, project-wide)
- **Lean utilities**: Insert symbols, access tactic templates, manage proofs

Use the command palette for additional context-aware actions and quick navigation throughout the workspace.
